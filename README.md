# Vue.js Todo-Beispiel

Dieses Repository enthält eine einfache Beispielanwendung mit dem JavaScript-Framework Vue.js.
Die App lädt Todo-Items von der öffentlichen JSON-API `https://jsonplaceholder.typicode.com/todos` und zeigt grundlegende Vue-Konzepte:

- Anzeigen einer Liste mit `v-for`
- Datenbindung mit `{{ }}` und `v-model`
- Event-Handling mit `@click`
- Dynamische CSS-Klassen für erledigte Todos
- Löschen einzelner Todos und Entfernen aller erledigten Todos

Die Oberfläche ist mit Bootstrap 4 gestaltet.

## Verwendete Technologien

- HTML5
- [Vue.js 2](https://vuejs.org/) (per CDN)
- [Bootstrap 4.3.1](https://getbootstrap.com/docs/4.3/getting-started/introduction/) (per CDN)
- Browser-Fetch-API
- Öffentliche JSON-API: `https://jsonplaceholder.typicode.com/todos`

## Nutzung

1. Repository klonen oder Dateien herunterladen.
2. `index.html` im Browser öffnen (Doppelklick oder via lokalem Webserver).
3. Todos ansehen, abhaken, löschen oder alle erledigten entfernen.

# Vue.js Todo Example

This repository contains a simple sample application built with the Vue.js framework.
The app loads todo items from the public JSON API `https://jsonplaceholder.typicode.com/todos` and demonstrates core Vue concepts:

- Rendering a list with `v-for`
- Data binding with `{{ }}` and `v-model`
- Event handling using `@click`
- Dynamic CSS classes for completed todos
- Deleting single todos and removing all completed todos

The UI is styled with Bootstrap 4.

## Technologies

- HTML5
- [Vue.js 2](https://vuejs.org/) (via CDN)
- [Bootstrap 4.3.1](https://getbootstrap.com/docs/4.3/getting-started/introduction/) (via CDN)
- Browser Fetch API
- Public JSON API: `https://jsonplaceholder.typicode.com/todos`

## How to Run

1. Clone this repository or download the files.
2. Open `index.html` in your browser (double-click or use a local web server).
3. Browse, toggle, and delete todos or remove all completed ones.
